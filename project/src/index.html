Балистические шлема

Военный баллистический шлем с тактическими вивер планками Ops-Core. Колокол шлема выполнен из СВМП, внутри установлены мягкие амортизирующие подушки на велкро. Имеет интегрированное крепление для установки прибора ночного видения и боковые направляющие, для установки дополнительного оборудования: нашлемного фонаря, экшн-камеры или креплений для очков, маски. На панелях с липучкой "велкро" можно носить различные нашивки и опознавательные знаки.
